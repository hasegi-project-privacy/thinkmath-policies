# thinkmath-policies

생각수학(Saenggak Math) 앱의 이용약관 및 개인정보 처리방침 호스팅 리포지토리.

## URLs (GitHub Pages)

- 메인: https://hasegi-project-privacy.github.io/thinkmath-policies/
- 이용약관 (한국어): https://hasegi-project-privacy.github.io/thinkmath-policies/terms-of-service-ko.html
- 개인정보 처리방침 (한국어): https://hasegi-project-privacy.github.io/thinkmath-policies/privacy-policy-ko.html

## 핵심 원칙

이 앱은 별도의 서버를 운영하지 않으며, **사용자의 개인 식별 정보를 수집하지 않습니다**.
모든 학습 데이터(오답노트, 손글씨, 첨부 이미지, 학습 기록, 설정)는 **기기 내부 저장소에만 저장**됩니다.
앱 삭제 시 모든 로컬 데이터가 함께 삭제됩니다.

광고 제공을 위해 Google AdMob SDK가 포함되어 있으며, 광고 식별자 등 일부 정보가 Google의 정책에 따라 처리됩니다.
