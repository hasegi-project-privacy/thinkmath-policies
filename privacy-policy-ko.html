생각수학 개인정보처리방침

시행일자: 2026년 04월 24일

생각수학은 수학 개념 탐색과 오답노트 학습을 지원하는 교육용 모바일 앱입니다. 본 앱은 별도의 서버를 운영하지 않으며, 모든 학습 데이터는 사용자 기기의 내부 저장소에만 저장됩니다. 개인정보보호법을 준수하여 사용자의 정보를 안전하게 관리합니다.

제1조 (개인정보의 처리목적)

생각수학은 다음의 목적을 위해서만 최소한의 정보를 처리합니다:

  1. 학습 서비스 제공
    • 수학 개념 탐색 및 학습 진행 상태 저장
    • 오답노트 작성·관리·복습 기능 제공
    • 손글씨(필기) 및 첨부 이미지 저장
  2. 앱 기능 제공
    • 학습 리마인더 푸시 알림 제공
    • 앱 내 설정 및 사용 환경 저장
  3. 광고 및 서비스 운영
    • Google AdMob을 통한 모바일 광고 노출
    • iOS 앱 추적 투명성(ATT) 동의 관리

제2조 (수집하는 개인정보의 항목 및 수집 방법)

1. 앱이 직접 수집·저장하는 정보

생각수학은 회원가입이나 로그인 절차가 없으며, 이름·이메일·전화번호 등 개인 식별정보를 수집하지 않습니다.

다음 정보는 오직 사용자의 기기 내부 저장소(로컬)에만 저장되며, 외부 서버로 전송되지 않습니다:

  • 학습 진행 기록 (풀이한 개념, 진도, 즐겨찾기 등)
  • 오답노트 내용 (문제, 풀이, 메모)
  • 사용자가 직접 촬영하거나 앨범에서 선택한 이미지
  • 사용자의 손글씨(서명·필기) 데이터
  • 앱 설정값 (알림 시간 등)

2. 자동 수집되는 정보 (제3자 SDK)

앱에 포함된 Google AdMob SDK는 광고 제공을 위해 다음과 같은 정보를 자동으로 수집할 수 있습니다:

  • 광고 식별자(IDFA/AAID)
  • 기기 정보 (OS 버전, 기기 모델, 언어, 국가 등)
  • 대략적 위치 정보 (IP 주소 기반)
  • 앱 사용 로그 및 광고 상호작용 정보

위 정보는 Google의 개인정보처리방침에 따라 처리되며, 사용자는 iOS의 앱 추적 투명성(ATT) 설정 또는 Android 광고 설정을 통해 광고 개인화를 제어할 수 있습니다.

제4조 (개인정보의 제3자 제공)

생각수학은 사용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외로 합니다:

  • 사용자가 사전에 동의한 경우
  • 법령의 규정에 의거하거나, 수사 목적으로 법령에 정해진 절차와 방법에 따라 수사기관의 요구가 있는 경우

제5조 (개인정보처리의 위탁)

생각수학은 별도의 서버를 운영하지 않으므로 개인정보 처리 업무를 위탁하지 않습니다. 다만, 다음의 제3자 서비스 제공자(SDK)를 앱에 포함하고 있습니다:

업체명 제공 서비스 관련 정책
Google LLC Google AdMob (광고 제공) Google 개인정보처리방침
Google LLC Google Mobile Ads SDK / 광고 식별자 Google 광고 정책

제6조 (앱 권한 안내)

생각수학은 다음과 같은 기기 권한을 사용합니다. 각 권한은 해당 기능을 사용할 때만 요청되며, 거부하더라도 해당 기능을 제외한 앱의 다른 기능은 정상적으로 이용하실 수 있습니다.

  • 사진/앨범 접근: 오답노트에 이미지를 첨부하기 위해 사용
  • 카메라: 오답노트 작성 시 문제를 직접 촬영하기 위해 사용
  • 저장공간: 오답노트 및 학습 데이터를 기기에 저장하기 위해 사용
  • 알림: 학습 리마인더 푸시 알림을 제공하기 위해 사용
  • 앱 추적 투명성(iOS): 맞춤형 광고 제공 여부 확인

제7조 (정보주체의 권리·의무 및 행사방법)

사용자는 언제든지 다음과 같은 방법으로 자신의 학습 데이터를 관리할 수 있습니다:

  1. 앱 내에서 오답노트 및 학습 기록을 개별 삭제
  2. 앱 내 설정에서 알림 권한 해제
  3. 기기 설정에서 광고 식별자 초기화 또는 맞춤형 광고 비활성화
  4. 앱 삭제(Uninstall)를 통한 모든 로컬 데이터의 일괄 삭제

제8조 (개인정보의 파기)

사용자가 앱 내에서 데이터를 삭제하거나 앱을 제거하는 즉시, 해당 데이터는 기기에서 영구적으로 삭제됩니다. 생각수학은 별도의 서버를 운영하지 않으므로 서버에 보관된 개인정보는 존재하지 않습니다.

제9조 (개인정보의 안전성 확보조치)

생각수학은 수집된 개인정보의 안전한 관리를 위해 다음과 같은 조치를 취하고 있습니다:

  • 기술적 조치: 학습 데이터를 기기 내부 저장소(앱 샌드박스)에만 저장하여 외부 접근을 차단
  • 최소 수집 원칙: 서비스 제공에 필요한 최소한의 정보만 사용
  • 제3자 SDK 관리: 검증된 SDK(Google AdMob 등)만 사용하며, 해당 업체의 개인정보 정책을 준수
